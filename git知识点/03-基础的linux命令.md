

### Git 底层概念（底层命令）

#### 基础的Linux命令

- **clear**： 清除屏幕
- **echo 'test content'**：往控制台输出信息 echo 'test content' > test.txt
- **ll**：将当前目录下的 子文件&子目录平铺在控制台
- **find 目录名**：将对应目录下的子孙文件&子孙目录平铺在控制台
- **find 目录名 -type f**：将对应目录下的文件平铺在控制台
- **rm 文件名**：删除文件
- **mv 源文件 重命名文件**：重命名
- **cat 文件的url**：查看对应文件的内容
- **vim 文件的url（在英文模式下）**
  - 按 i 进入插入模式  进行文件的编辑
  - 按 esc 键 & 按 : 键   进行命令的执行
  - q!     强制退出（不保存）
  - wq   保存退出
  - set nu  设置行号
